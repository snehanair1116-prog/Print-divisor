# Print-divisor
 The Logic: "The Square Root Pair"Most people loop from 1 all the way to n. But divisors always come in pairs.If n = 36:2 is a divisor, and 36 / 2 = 18. So 18 is also a divisor.6 is a divisor, and 36 / 6 = 6. So 6 is a divisor (a perfect square).By only looping up to sqrt{n}, we can find both numbers in the pair at the same time!
